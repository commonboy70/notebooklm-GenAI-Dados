# NotebookLM Chat – Curso X

## Descrição
Este repositório contém o registro completo de um chat realizado no **Google NotebookLM** para estudo e entrega no curso X. Ele inclui:

- Perguntas feitas durante a conversa.  
- Respostas geradas pelo NotebookLM.  
- Arquivos extras (PDF ou notas) para referência complementar.

O conteúdo foi organizado para facilitar a leitura, revisão e uso acadêmico.

## Arquivos do repositório
- `perguntas.md` → todas as perguntas feitas no chat  
- `respostas.md` → respostas completas do NotebookLM  
- `extras.pdf` → exportação do chat em PDF (opcional)  

## Link do chat original
Acesse o NotebookLM aqui: [Chat NotebookLM](https://notebooklm.google.com/notebook/bbf1c278-03d4-429e-a06e-1cdfb35f4118)

## Como usar
1. Abra os arquivos `perguntas.md` e `respostas.md` para ler a conversa completa.  
2. Consulte `extras.pdf` para ver a exportação completa (opcional).  
3. Este repositório pode ser usado como material de estudo ou referência em trabalhos do curso.
